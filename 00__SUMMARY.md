Этот файл — единая точка входа для ассистента.
GitHub используется только как read-only источник актуальных Markdown/JSON по **RAW-ссылкам**.

## START (обязательно в начале каждого нового чата)
1) Открыть этот `00__SUMMARY.md`.
2) Открыть `Assistant Guides/00__SUMMARY.md` и быстро освежить правила/процесс.
3) Дальше навигировать по summary-дереву: сначала `00__SUMMARY.md` нужной папки, затем конкретные файлы.

## Короткие правила навигации
- Всегда предпочитай **RAW** (`raw.githubusercontent.com`) вместо `github.com/.../blob/...`.
- Сначала читаем SUMMARY, затем точечно открываем файлы по RAW-ссылкам.
- Если ссылки/summary не хватает — попроси у пользователя RAW-ссылку на нужный summary/файл или обнови соответствующий summary.
- ZIP-бандл выдаётся **только по явной просьбе** «собери бандл / дай бандл».
- Удаление/перенос/переименование файлов делается через manifest (см. гайд `06__Manifest_delete_move.md`).
