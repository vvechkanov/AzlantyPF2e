# AZL — ROOT SUMMARY

Этот файл — единая точка входа **для ассистента**.
GitHub используется только как read-only источник текста по RAW-ссылкам.

## START (в начале каждого нового чата)
1) Открыть этот ROOT SUMMARY.
2) Открыть `Assistant Guides/00__SUMMARY.md` и при необходимости перечитать правила.
3) Дальше ходить по summary-дереву: сначала `00__SUMMARY.md` нужной папки, затем точечно открывать файлы по RAW-ссылкам.
