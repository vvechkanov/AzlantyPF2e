# AZL — доступ к Project Knowledge (GitHub)

Этот проект хранит знания (канон/сводки/summary/файлы) **вне Project Files**, в GitHub, в виде Markdown/JSON.
GitHub используется **только для ассистента (read-only)**. Obsidian — основной источник для человека.

## Правила
- Всегда открывай файлы по **RAW-ссылкам** (`raw.githubusercontent.com`), не по `github.com/.../blob/...`.
- Не используй ссылки с `?token=...` (токены не нужны для публичного чтения).
- Если в индексе нет нужной ссылки — ориентируйся по summary-дереву и/или запроси у пользователя raw-ссылку на нужный summary/файл.
