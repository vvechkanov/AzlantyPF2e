# 06 — Manifest: delete / move (если используем)

Обновлено: 2026-01-11

Manifest нужен только для операций:
- удалить файл
- перенести/переименовать файл

Минимальный пустой:
```json
{"version":1,"ops":[]}
```

Рекомендуемый формат ops:
```json
{
  "version": 1,
  "ops": [
    { "op": "delete", "path": "Сюжет кампании/Арка-1/Старый.md" },
    { "op": "move", "from": "A/Было.md", "to": "A/Стало.md" }
  ]
}
```

Правила:
- операции должны быть явными (не “удалить потому что файла нет”)
- после move/rename обновляем summary в обеих затронутых папках
- после delete обновляем summary папки, где файл был
