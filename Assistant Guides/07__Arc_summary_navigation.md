# 07 — Как находить SUMMARY арок и подпапок (navigation)
Обновлено: 2026-01-12

Этот гайд — для ассистента. Цель: быстро и без гаданий находить **нужный `00__SUMMARY.md`** и дальше — каноничные файлы по RAW-ссылкам.

## Правило навигации (всегда одно и то же)
1) Открыть **ROOT SUMMARY**: `00__SUMMARY.md`
2) Перейти в нужную верхнюю ветку контента (чаще всего это `Сюжет кампании/00__SUMMARY.md`).
3) Внутри неё найти подпапку (например `Арка-1/`) и открыть её **`00__SUMMARY.md`**.
4) Дальше идти **только по RAW-ссылкам**, указанным в `00__SUMMARY.md` этой папки.

## Шаблон для сюжетных арок
- ROOT SUMMARY → ветка **Сюжет кампании** → `Сюжет кампании/00__SUMMARY.md`
- Там: `Арка-N/` → raw-ссылка на `Сюжет кампании/Арка-N/00__SUMMARY.md`
- В `Арка-N/00__SUMMARY.md` должны быть RAW-ссылки на файлы арки.

Пример (Арка‑1):
- `Сюжет кампании/00__SUMMARY.md` → raw: https://raw.githubusercontent.com/vvechkanov/AzlantyPF2e/master/%D0%A1%D1%8E%D0%B6%D0%B5%D1%82%20%D0%BA%D0%B0%D0%BC%D0%BF%D0%B0%D0%BD%D0%B8%D0%B8/00__SUMMARY.md
- `Сюжет кампании/Арка-1/00__SUMMARY.md` → raw: https://raw.githubusercontent.com/vvechkanov/AzlantyPF2e/master/%D0%A1%D1%8E%D0%B6%D0%B5%D1%82%20%D0%BA%D0%B0%D0%BC%D0%BF%D0%B0%D0%BD%D0%B8%D0%B8/%D0%90%D1%80%D0%BA%D0%B0-1/00__SUMMARY.md

## Если `00__SUMMARY.md` арки “пустой” или без RAW-ссылок
Ассистент **не гадает** и не придумывает содержимое.

Действия:
1) Внести правку: добавить список файлов/подпапок **с RAW-ссылками** (как минимум на те файлы, которые уже используются/упоминаются).
2) Если `00__SUMMARY.md` арки отсутствует — создать его.
3) Обновить `00__SUMMARY.md` на уровень выше (чтобы ссылка на арку была доступна из дерева).

## Если файл выглядит “обрезанным”
Иногда просмотрщик может показать только фрагмент файла. Решение:
- **скачать RAW целиком и читать локально**, затем уже делать выводы/правки.

## Формат, чтобы summary читался стабильно
- одна ссылка/элемент — одна строка (список `- ...`), без гигантских “однострочников”.
